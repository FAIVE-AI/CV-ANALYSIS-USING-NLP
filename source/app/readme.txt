Instructions for Running the React App on your Computer:

1. Install Node JS from https://nodejs.org/en/download if not already installed
2. Restart your computer if Node is newly installed - else skip (not required always but better to do to allow environment variable entries)
3. Open the 'source' folder within the local repository
4. In Visual Studio Code - Open terminal (Windows: Ctrl + ~) (Mac: Ctrl + `)
5. In the terminal change directory to 'app' folder
6. In the terminal type npm install and wait for all the packages to install
7. Once done, type npm run start and the website will load in your default browser
8. Make code changes and save changed files to see updates on the browser
